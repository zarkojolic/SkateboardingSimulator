# Skateboarding Gameplay Prototype (UE 5.5)

Controls

- **W** – Move Forward  
- **A / D** – Steer Left / Right  
- **Space** – Jump  
- **Left Shift** – Push  
- **S** - Brake
- **Mouse** – Camera look

Build Information

- Engine version: **Unreal Engine 5.5**
- Platform: **Windows (64-bit)**

The packaged build (**.exe**) is **not included in this repository**  
as per instructions (to avoid unnecessary repository size).

The Windows executable build is provided **separately** as a ZIP file  
and should be downloaded and run locally for testing, folder where build
is located is called "SkaterSimBuild" in the SkateboardingSim main folder.
